# stevedores.org

Open source AI packaging platform. Package AI agents into production-ready containers with [oxidizedgraph](https://github.com/stevedores-org/oxidizedgraph).

## Development

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
```

## Stack

- **Runtime**: Bun
- **Framework**: React + TypeScript
- **Build**: Vite
- **Styling**: Tailwind CSS v4
